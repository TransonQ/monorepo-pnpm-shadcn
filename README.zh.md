## 脚本

在根目录执行如下命令用于创建新项目,可以根据需求在`templates/*`中定制自己的模板.

```bash
# 根据模板 vite-app 在 apps/* 工作去创建一个新项目
pnpm run new-app
```

案例:

```bash
pnpm run new-app

✔ Please enter the project name: my-app

```
